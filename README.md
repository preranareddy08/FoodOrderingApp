# FoodOrderingApp
 Developed a dynamic Food Ordering App using React, constructed entirely from scratch without the aid of Create
React App.
– This application leverages live Swiggy APIs to provide real-time data on a wide range of restaurants, menus, and
food items.
– Key aspects of this app include React development techniques, efficient API integration, and meticulous state
management with hooks.
Technologies used : React Js, Tailwind CSS, REST API Calls
